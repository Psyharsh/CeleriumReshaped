# CeleriumReshaped
A performance and stability-focused modpack that is built on the Celerium foundation.

# Modrinth description:

# Now, Meet Celerium: Reshaped!
Minecraft, but more refined.
Designed for Fabric 1.21.11.
Updates will prioritize stability over speed.

## Overview
This modpack revolves around one idea: a pack that focuses on performance, stability, and improvements that feel like they **belong** in the game. The result is a modern Fabric experience that stays true to vanilla, while quietly fixing what vanilla leaves behind.

## What makes it different

Most modpacks try to do more.
This one focuses on doing less, better.

- No feature bloat
- No overlapping mods
- No intrusive UI changes
- No unnecessary mechanics

Every mod exists for a reason, and if two mods solve the same problem, only the best one stays.

## Choose Your Experience

Celerium is available in three variants. Each one is built on the same foundation — the difference is how much is added on top.

### CORE — Full Experience
- All 41 mods active
- Balanced configuration for performance + usability
- Includes QoL, HUD, and visual improvements

Best for: Most players. This is the intended Celerium experience.

### LITE — Performance Focused
- Same mod list as CORE
- Configurations tuned for lower-end hardware
- Reduced visual intensity and background load

Best for: Older PCs or players who want smoother performance without losing features.

### MINIMAL — Vanilla-Like
- Only essential mods
- No QoL, HUD, or visual extras
- Clean, lightweight, near-vanilla feel

Best for: Players who want maximum performance or a mostly vanilla experience with essential improvements.

### Important

All versions share the same core optimizations and stability improvements.

MINIMAL removes convenience features — not the underlying performance foundation.

## Structure

The pack is organized into clearly defined layers:

- Core
  - Fundamental systems and APIs that shape the experience
- Performance
  - Engine-level optimizations to ensure smooth gameplay on modern systems
- Stability & Fixes
  - Bug fixes, crash prevention, and vanilla corrections
- Essential Quality of Life
  - Subtle improvements that feel like they should already be part of Minecraft
- Libraries
  - Required dependencies that keep everything running cleanly behind the scenes
 ## Optional Content

Additional features are provided as modular extensions, so you can customize your experience without affecting the core design:

- Visual Enhancements
  - Improved visuals that respect Minecraft’s original style
- Quality of Life (Extended)
  - Extra conveniences for players who want more flexibility
- Shader Support
  - Optional compatibility for shader-based rendering
> Said optional mods not available in the minimal edition.

These are entirely optional — the base pack is designed to feel complete on its own.

## The Goal

To provide a version of Minecraft that feels:

- Smoother
- Cleaner
- More consistent
- More complete

Without ever feeling like something else.
